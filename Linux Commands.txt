===> pwd[Print Working Directory] ---> It shows your current location (directory) in the filesystem.
===> cd ---> Change Directory. It changes the current directory to the specified directory

===> ls[List] ---> Lists the files and directories in the current directory
===> ls -a ---> It is used to list all files and directories, including hidden files
===> ls -l ---> Lists the files and directories in long format
===> ls -a -l or ls -al or ls --all -l ---> It is used to list all files and directories with detailed information, including hidden files
===> ls -t ---> Lists with creation time sorted (recent at the top)
===> ls -r ---> Lists in reverse order
===> ls -lrt (OR) ll ---> Lists in long format, reverse order and time sorted

===> touch <file_name> ---> Creates an empty file or updates the timestamp of an existing file. [touch file.txt]
 
===> rm <file_name> ---> Removes(deletes) a file. 	[rm file.txt]
===> rm -r <directory_name> ---> Recursively removes a directory and it's contents. 	[rm -r Dir2]
===> rmdir <directory_name ---> Removes an empty directory

===> cd .. ---> 1 step back
===> cd ../.. ---> 2 step back

===> tree ---> It is used to display the contents of current directory in a tree-like format. It recursively lists all files and directories within a specified directory.
===> sudo apt update ---> Before installing any software/tools, refresh/update the package list to get the latest package
===> sudo apt install tree ---> Once the package list is updated, install the desired software using apt
Note: sudo apt update and sudo apt install tree is used to update and install tree
===> tree -d ---> Display Only Directories
===> tree -a ---> List All Files Including Hidden Files
===> tree -L 2 ---> Limit the Depth of the Tree 
===> tree -f ---> Display Full Path Prefix for Each File 
===> tree -h ---> Print the Size of Each File
