19 what is file Systern Hierarchy?
===> File System Hierarchy follows FHS [Filesystem Hierarchy System] where everything starts from root "/", and directories like /bin, /etc, /home, /var are
organised based on their purpose

Basic Linux Directory Structure
-------------------------------
		/
		|-bin
		|-boot
		|-dev
		|-etc
		|-home
		|-lib
		|-media
		|-mnt
		|-opt
		|-proc
		|-root
		|-run
		|-sbin
		|-tmp
		|-usr
		|-var

/(Root)---> Top level directory. Everything starts here
/(bin) ---> Essential user Commands.	Ex:- ls, cp, mv, cat, grep
/sbin ---> System/admin commands. 		Ex:- reboot, fdisk, ifconfig
/etc ---> configuration files
/home ---> Home directories. i. e /home/ubuntu
/var ---> Variable data [Changes Frequently]		Ex:- /var/log, /var/lib
/tmp ---> Temporary files
/usr ---> User Programs and Utilities.			Ex: /usr/bin, /usr/lib, /usr/local
/Proc ---> virtual filesystem showing system info.		Ex: /proc/cpuinfo, /proc/meminfo
